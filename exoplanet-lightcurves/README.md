# Exoplanet Detection by Light Curve Analysis

## Project Goal
Analyze photometric data from NASA's Kepler Telescope's observation of Kepler-10, a confirmed multi-planet system.

## Method
- Data Source: NASA Kepler mission via Lightkurve Python Library
- Target Star: Kepler-10
- Technique: Transit photometry - detecting dips in stellar brightness

## Resources
- Python, Lightkurve, Matplotlib, Numpy

## Status
v1 - Single star analysis (Kepler-10)
